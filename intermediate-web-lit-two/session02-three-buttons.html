Intermediate Web Literacy II | Storytelling with Scripts | Storytelling with Three Buttons

Made by Mozilla

30 to 45 minutes

Learn how HTML, CSS, and JavaScript connect to one another by using buttons and scripts to tell a simple story.

Write the Web

21st Century Skills

  • Communication
  • Creativity

Web Literacy Competencies

Code Compose Remix

Learning Objectives

  • Understand how CSS stylesheets and JavaScript code connect to an HTML webpage.
  • Code multiples button and a script that work together to tell a story inside through alert windows.

Audience

  • 13+
  • Intermediate web users

Materials

  • An Internet-connected, projected instructor computer
  • Internect-connected computers
  • Markers
  • Paper

  • 1. Preparation

    Work through today's activity to get a feel for how each of its parts - the HTML, CSS, and JavaScript - looks and for how those parts work together. Review this lesson plan, as well, to make sure you're comfortable facilitating it.

    Be sure to test your technology and Internet connectivity.

    Come up with an equitable way for your learners to form groups of four.

    Create a class account or several accounts for students to use on Thimble, Mozilla's online coee editor, and post the URL, or a shortened link, for today's project somewhere highly visible in the learning space.
