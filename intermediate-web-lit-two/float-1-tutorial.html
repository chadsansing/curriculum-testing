Webotron & the Scrambled Divs Tutorial

This project will help you learn the rules of clear and float, two attributes people use to create dynamic and responsive layouts through CSS. As you follow the tutorial you will

  1. Know and understand how to use the clear and float.
  2. Change the order of div elements on webpage using HTML.
  3. Change the float value of div elements using CSS.
  4. Change the clear value of div elements using CSS.
