  3. Use CSS to change the background-image and appearance of a div element syled like a box on a webpage.
  4. Stack multiple CSS values to create larger text and box effects on an HTML webpage.

Step-by-Step Instructions

  1. How do webpages and stylesheets connect?

    TL;DR: HTML webpages use the link element to call in instructions from external CSS stylesheets.

    If you check out line 21 of index.html, you can see how that webpage connects to style.css with a link. The link takes an href, or 'hypertext reference,' attribute with the stylesheet's home address as its value. Some stylesheets, like style.css, live in the same folder alongside index.html, so you just need to type in the file's name.

    You can also use the link element to call in external stylesheets from the web. In fact, many of the fonts used on index.html are webfonts called in with the stylesheet links between line 13 and line 19.

    The link acts like a tunnel or passageway for the instructions from the stylesheet to enter the webpage and 'cascade' downwards, telling each subsequent part of the page how to look.

    Back to navigation.

  2. What's text-decoration?

    TL;DR: You can use the text-decoration property to underline, overline, or strikethrough text.

    You can use the text-decoration property in a stylesheet instead of the HTML tag <u> (for 'underline') to underline text. You can also overline or strike-through text using text-decoration. However, to make a text bold or italic with CSS you have to play around with the font-family and font-weight properties or else use <strong> or <em> tags right in your HTML. The strong tag makes text bold, while the em tag makes text italic using 'emphasis.'

    Back to navigation.

  3. What's text-transform?

    TL;DR: You can use the text-transform property to control the capitalization in any passage of text.

    If you want to draw attention to a particular passage of text or give it a definite mood, tone, or voice, you can use text-transform to make every letter in the passage lowercase or uppercase. You can also set the value of text-transform to capitalize to capitalize the first letter in each word of a passage of text.
