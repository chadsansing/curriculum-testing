CSS Pyramid Tutorial

This project will help you learn how to use the font-size attribute and CSS to change the size of default elements, like h1, as well as the size of unique elements (that use ids) and type of elements (called classes). By the time you finish, you will

  1. Know and understand how to use a stylesheet to style parts of a webpage.
  2. Know and understand how to style font-size using CSS, or a cascading style sheet.
  3. Change font-size of default HTML elements like h1 and h2.
  4. Change font-size of unique HTML elements with id labels.
  5. Change font-size of types of HTML elements with class labels.

Step-by-Step Instructions

  1. What's CSSs?

    TL;DR: CSS tells your broswer how to show different parts of your webpage.

    According to the Mozilla Developer Network (MDN),a Cascading Style Sheet

    most of the time abbreviated as CSS, is a stylesheet language used to describe the presentation of a document written in HTML.... CSS describes how the structured element must be rendered on screen, on paper, in speech, or on other media.

    What does that mean?

    CSS gives your broswer instructions about how to display each part of a webpage. Browsers have default settings, so if you don't give a webpage CSS instructions, a browser will just show your text and media in a plain serif font on a white background. However, if you include a stylesheet in your code (using the <script></script tags) or if you link your webpage to an external stylsheet (saved as a .css file), you can control how your page looks. For example, you can set a particular kind of font, as well as the size and color of the letters on your page. Advanced CSS users can even animate parts of a page using display cues and timings.

    When your write a stylesheet, it's like you're creating a text-based paint-by-numbers for a browser to follow while displaying your webpage.

    You can see the link that connects our webpage, index.html, to our stylesheet, style.css at line 14 of the index.html file in the left sidebar called FILES. It looks like this:

    <link rel="stylesheet" href="pyramid-1-style.css" type="text/css">

    If you click on style.css, you can see the stylesheet itself. The directions from the stylesheet make it into the webpage through that link at line 14.

    Back to navigation.

  2. What's an id?

    TL;DR: An id is a unique label you can give an HTML element to style it differently from other parts of your webpage.

    According to the Mozilla Developer Network (MDN), id

    defines a unique identifier (ID) which must be unique in the whole document. Its purpose is to identify the element when linking (using a fragment identifier), scripting, or styling (with CSS).

    What does that mean?

    When you use an id, you give part of your page an absolutely unique identity. You can style this identity with specific instructions in your stylsheet that apply only to the element with the matching id.

    It's useful to use an ID when you have to position a partof your page just-so or when you want part of your page to stand out from the rest because of an attrbiute like color or size.

    Here is one way to think of how id relates to the rest of your webpage and stylesheet, including parts labeled with a class:

    • Imagine that all people are like a default element (say paragraphs or <p> tags) in HTML.

    • Some of those people might belong to a certain class, like, 'People in the band.'
    • One person might have a specific role or id in that band, like, 'The singer.'

    You can see an example of an id and its styling instructions by looking at the div called levelOne at line 32 of index.html and line 41 of style.css.

    HTML:<div id="levelOne" class="textRow"></div>

    CSS:
    #levelOne{
    font-size: 2em;
    }

    Back to navigation.

  3. What's a class?

    TL;DR: A class is a shared label you can apply to several parts of a webpage to display them the same way.

    According to the Mozilla Developer Network (MDN),

    is a space-separated list of the classes of the element. Classes allows CSS and Javascript to select and access specific elements via the class selectors....

    What does that mean?

    When you use a class in your stylesheet and webpage, you tell your broswer to display every element with that class in the same way. For example, if you created a class called redText, you could give your stylesheet instructions to display every redText paragraph or heading with a red font color.

    You can see an example of a class and its styling instructions by looking at a div with labeled with textRow at line 32, line 34, or line36 of index.html and line 62 of style.css.

    HTML:<div id="levelOne" class="textRow"></div>

    CSS:
    .textRow{
    color:#303;
    font-family: 'Fredoka One', cursive;
    text-align:center;
    text-shadow: -1px -1px #aaa;
    }

    Back to navigation.

  4. What is font-size

    TL;DR: You can use CSS to control the size of text on a webpage.

    According to the Mozilla Developer Network (MDN), font-size

    specifies the size of the font. Setting the font size may, in turn, change the size of other items, since it is used to compute the value of the em and ex units.

    What does that mean?

    When you set a value for font-size, you tell broswers how big or small to make the text on your webpage. You can give different types of text - like headers and paragraphs - different font sizes, and you can use id and class to give even more specific font-size instructions to different parts of your page.

    Here, em is a unit of measurement of font-size, with broswers' default paragraph text set to 1em. You can also set font-size in pixels, or px, with a default value of something like 12px.

    If you look at line 31, line 35, and line 62 of style.css, you can see different font-size values for parts of our webpage like h1, h2, and .textRow.

    Back to navigation.

  5. Change the font-size

    TL;DR: Change the font-size of each .textRow to invert the pyramid of words.

    Go to line 41 of style.css. Change the font-size values of the div elements (each labeled with an id) called levelOne, levelTwo, and levelThree to invert the pyramid and make the top line the biggest and the bottom line the smallest.

    Back to navigation.

    TL;DR: You can mess with any line of code you want.

For example, you can see what happens when you change the background-color of the body element or the color of text tagged with h2 or h2. Play around and see what you can learn!
