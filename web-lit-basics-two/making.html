The Web Is for Making

Try a project and make something with the Web!

Beginner

  • Build a giant cardboard marble maze with Instructables.
  • Caption an image to remix a Meme with DIY.org.
  • Launch this foam falcon with Make.

Intermediate

  • Assemble this cardboard mini-foosball game from Make .
  • Make a cardboard automaton with the Exploratorium.

Advanced

  • Watch "Turn your Smartphone into a 3D Hologram," and then see how to recycle plastic packaging so you don't have to find and cut up a CD case.

Younger makers: Make sure you get adult help with all cutting tools.
