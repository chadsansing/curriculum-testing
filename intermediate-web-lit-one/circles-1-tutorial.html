This project will help you learn style the color, position, and style of content on a webpage. You will

  1. Know and understand how to use the <div> tag.
  2. Change the color of circles made using the <div> element.
  3. Change the size of those circles.
  4. Change the position of those circles.

Step-by-Step Instructions

  1. What's a div?

    TL;DR: a div is an element you can use to create part of a webpage with special rules.

    According to the Mozilla Developer Network, a div is

    the generic container for flow content, which does not inherently represent anything. It can be used to group elements for styling purposes (using the class or id attributes), or because they share attribute values, such as lang. It should be used only when no other semantic element (such as <article> or <nav>) is appropriate.

    What does that mean?

    Basically, it means that when you create a div and give it an id or class (a sepcial label), you can write instructions on a cascading style sheet (CSS) that create special rules for how that div displays on a webpage. You can mess with its color, position, and size, to begin with, as well as any special effects you want the div to have, like animations or shadows, that are different from the rest of the page.

    If you want to create a sort of unique part of your web page and no other element like header, p (for paragraph), or section will do, then use a div.

  3. Change background-color and border color

    TL;DR: you can use color codes and names to change the color of elements on a webpage.

    There are plenty of ways to change the color of elements on a webpage using .css files. If you are in a 'TL;DR' mood, just mess around with color names and hexcodes today, and skip the last section on rgba colors.

    Websafe color names

    To begin, go to our style.css file and visit the div elements representing our circles at the bottom of the style page.

    Each circle div has two colors, a background-color attribute and a color value for its border attribute.

    You can change colors three ways with HTML and CSS. First, you can just type 'websafe' color names, like 'red,' 'green,' or 'blue.' Here is a site from w3schools.com with color names (and hex codes, which we'll get to in a moment). Try typing some of these names into the CSS instructions for our circles. When you delete the color codes that are already there for background-color and the last part of each border element, be sure to delete the #, as well.

    Hexcodes

    Next, let's play with hexcodes. Color hexcodes (from the Greek 'hex' for 'six') are six-character color codes that look like this: #000000.

    In a color hexcode, the first two characters are red values, the next two characters are green values, and the last two characters are blue values. Characters can be any number between 0 and 9, or any letter between 'a' and 'f.' The number 0 is the darkest value, and the letter 'f' is the brightest value, so #000000 displays as black, while #ffffff displays as white. Something like #ff0000 would be a bright red, while #550000 would be a darker red. If ever you have the same two values for each color in a hexcode, like in #ff0000 where both red values are 'f,' both blue values are 0, and both green values are 0, you can shortcut the code down to three characters like this: #f00. That is to say, #ff0000 equals #f00. Unless you're building a webpage for printing, it's good to use hexcodes.

    Use a site like that w3schools.com list of colors or a chart like this one to pick hexcodes and mess with the background-color and border color value for each circle.

    RGBA values

    Finally, we can also use 'rgba' values for a color. The abbreviation 'rgba' stands for, 'red, green, blue, alpha,' where 'alpha' is a value for opacity. All rgba values are between 0 and 255, which represent the 256 steps (0 is the first step) a color can take in a 8-bit binary number (and these steps match those availble in a 5 volt circuit). Circuitry runs on those 8-bit binary numbers as electricity defines how much of a circuit is on or off. Those 8-bit binary numbers control the color display and intensity of electronics like multi-color LEDs or color-printer heads, so when you use rgba values, you're essentially borrowing analog, physical color values created by passing current through part of a circuit that creates a physical response to the amount of voltage it receives. Hence, 'rgba' values are great to use on webpages that see frequent printing or that want to approximate the colors of, say, an LED connected to a microprocessor like the arduino.

    In our style.css file, you can use rgba values like this: rgba(0, 0, 256, 1). In an rgba code, the first number represents the red value, the second is the green, the third is the blue, and the fourth is opacity. Colors can go from 0 (darkest) to 255 (brightest), and alpha, or opacity, can go from 0 (transparent) to 1 (opaque) using decimal values, as well.

    Here is a great rgba color resource that gives you the CSS code for background-color and color.

  5. Change position

    TL;DR: You can place an element like a div anywhere on a webpage using its stylesheet.

    As you take one more look at our circlees' div elements inside style.css, you can see that each has a position. The position of each div element works with its top attribute on this page. For example, an element like #circleOne that is positioned as relative with a top value of 25px is relatively positioned 25 pixels underneath the preceding element of the page - the beginning of the main layout element.

    There are several values for the position attribute. Here are three commonly used values:

    • Absolute - this kind of positioning anchors an element at a specific place, but still allows for page scrolling. Elements positioned with absolute depend on the entire page.
    • Fixed - this kind of positioning anchors an element at a specific place and anchors the whole page at the same time, preventing scrolling. Elements positioned with fixed depend on the entire page.
    • Relative - this kind of positioning places an element relative to the part of the page it's in or to the last element that appears before it. Elements positioned with relative depend on spatial relationships to other parts of the page rather than to only the entire page.

    Once you choose a value for position, you can place an element using top, bottom, left, or right. You can use either top or bottom and either left or right. You can also position something just from the top or bottom or just from gthe left or right.

    Dive back into style.css and see what you can 'break' by changing the value of position to absolute or fixed. You can also increase or decrease the values for top. Finally, you can also see what happens when you change top to bottom or left to right.

    You can also shift the blocks to line up horizontally instead of stacking vertically.

    1. Give each circular div a absolute value for position, as well as the same value for top.
    2. Next, add a left:_px; line to the instructions for each of those div elements on style.css.
    3. Fill in the blank with different pixel values to begin moving the circles around the page horizontally.

    You can play around with positioning even further by using a absolute or fixed value for position and then changing the circles' top and left values indepdently of one another to make a huge Venn diagram.

    You can also add more circular div elements to index.html like this:

    <div id="circleFour"></div>.

    Then, for each circle you add on index.html, go back to style.css and create a new set of instructions for that div element. You can copy and paste the circle before your new one and then change its name from, say, #circleThree to #circleFour on style.css. After that, change the values for each attribute in the circle (like height or top) until the div displays how you want it to on your webpage.

Secret missions!

Did you know that you can make triangular divs, too? Check this out.

  1. Add <div id="triangle"></div> beneath the circles in the main layout element of index.html.
  2. Go to style.css and add instructions for #triangle underneath the instructions for the circular div elements.
  3. Instead of using one border attribute for each circle in style.css, use any three of the following border attributes:
    • border-top.
    • border-bottom.
    • border-left.
    • border-right.
  4. Give each border attribute a value of 100px to create some volume and a border style of solid. (Remember, each border attribute works like this: border: [width] [style] [color];.)
  5. Give two of the border attributes a value of transparent for color.
  6. Give the third and last border the color value you want your entire triangle to have.
  7. Then, style the rest of that div with a height and width of 0.
  8. Give the triangle a position value of relative about 25px from the top of the last element.
  9. By now, your instructions for the triangle on style.css should look something like this:

    #triangle{
      border-top: 100px solid transparent;
      border-left: 100px solid transparent;
      border-right: 100px solid #00a;
      height:0;
      position:relative;
      top:25px;
      width:0;
    }
  10. Go back to index.html to see what happened!

You can change color values or add values for attributes like opacity to play around further with the triangle.
