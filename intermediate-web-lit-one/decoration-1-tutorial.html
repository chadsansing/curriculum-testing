CSS DecorationTutorial

This project will help you use CSS to decorate text and div elements on an HTML webpage using box-shadow, text-shadow, text-decoration, and text-tranform. As you work through the tutorial, you will

  1. Know and understand how an HTML webpage and CSS stylesheet connect and interact.
  2. Use CSS to change the appearance of inline text on a webpage.
  3. Use CSS to changer the appearance of block elements on a webpage.
  4. Use CSS to change the background-image and appearance of a div element syled like a box on a webpage.
  5. Stack multiple CSS values to create larger text and box effects on an HTML webpage.

Step-by-Step Instructions

  1. How do webpages and stylesheets connect?

    TL;DR: HTML webpages use the link element to call in instructions from external CSS stylesheets.

    If you check out line 21 of index.html, you can see how that webpage connects to style.css with a link. The link takes an href, or 'hypertext reference,' attribute with the stylesheet's home address as its value. Some stylesheets, like style.css, live in the same folder alongside index.html, so you just need to type in the file's name.

    You can also use the link element to call in external stylesheets from the web. In fact, many of the fonts used on index.html are webfonts called in with the stylesheet links between line 13 and line 19.

    The link acts like a tunnel or passageway for the instructions from the stylesheet to enter the webpage and 'cascade' downwards, telling each subsequent part of the page how to look.

    Back to navigation.

  2. What's text-decoration?

    TL;DR: You can use the text-decoration property to underline, overline, or strikethrough text.

    You can use the text-decoration property in a stylesheet instead of the HTML tag <u> (for 'underline') to underline text. You can also overline or strike-through text using text-decoration.

    Commonly used values for the text-decoration property include underline, overline, line-through, and none. You can add a color after each initial value, as well as a line style, like wavy. A line of stling instructions like this

    text-decoration:underline red dotted

    Looks like this when displayed on a webpage.

    Moreover, you can apply multiple text-decoration styles to the same text. For example,

    text-decoration: underline overline;

    Looks like this when displayed on a webpage.

    To make a text bold or italic with CSS you have to play around with the font-family and font-weight properties or else use <strong> or <em> tags right in your HTML. The strong tag makes text bold, while the em tag makes text italic using 'emphasis.'

    Back to navigation.

  3. What's text-transform?

    TL;DR: You can use the text-transform property to control the capitalization in any passage of text.

    If you want to draw attention to a particular passage of text or give it a definite mood, tone, or voice, you can use text-transform to make every letter in the passage lowercase or uppercase. You can also set the value of text-transform to capitalize if you want the first letter in each word of a passage of text to be capitalized.

    Here is an example of text-transform in action. A stylesheet that looks like this

    text-transform:capitalize

    would have this effect on the webpage.

    Back to navigation.

  4. What's text-shadow?

    TL;DR: You can use text-shadow to style text inside HTML elements with a shadow.

    As you might suspect, the text-shadow property adds a shadow to the passage of text it styles. Each text-shadow has an x-offset and a y-offset, as well as optional color and blur radius values. A simple text-shadow looks like this on a stylesheet:

    text-shadow: 2px 2px #fff;

    And like this on a webpage.

    A text-shadow value like that would add a shadow cast 2 pixels to the right of a word and 2 pixels down in white. You can stack multiple shadows like this to create thicker shadows or multi-colored ones (using a color picker like this one). To stack shadows, you separate them using commas. For example, this...

    text-shadow: 2px 2px #0c5, 4px 4px #50c, 6px 6px #c50

    ...looks like this on a webpage.

    You can also add a blur radius valoue in bteween the y-offset and the color to turn the shadow into a kind of aura or cloud all around the word. A blurred text-shadow looks like this on a stylesheet:

    text-shadow: 5px 5px 10px #f00;

    And like this on a webpage.

    A shadow like that blurs, or smudges, its x- and y-offset shadow by 5 pixels. You can stack blurred shadows to create ndifferent text effects, like a neon glow.

    Back to navigation.

  5. What's box-shadow?

    TL;DR: You can use a box-shadow to style HTML elements like img and div with a shadow.

    The box-shadow property works just like text-shadow except that it styles block and container elements instead of the inline text inside of them. If you have an element like an image (img) or div or even a paragraph (p), you can use box-shadow to style a shadow around the outside boundaries of that element. For example, if we decided to create a box-shadow around an entire paragraph, we could use stylesheet language like this

    box-shadow: 10px 10px #fff;
